# Agent Skill Vault

Use this repository as a multi-domain skill source and MCP server.

For project work, install pinned skills into the target project first, then follow the target project's `CLAUDE.md`, `.claude/skills`, and `.agents/skills.lock.json`. If the lock requires AgentMemory, check the `agentmemory` MCP server first, search memory before planning, and save durable decisions when finishing meaningful work.
